Start to work with WebComponents. Everything that you need is ready to use.

## Launch the application

```sh
$ npm install
$ npm start
```

## Release in Production

```sh
$ npm run prod
```

## Tech Stack

### Libraries

- [@webcomponents/webcomponentsjs](https://github.com/webcomponents/webcomponentsjs)
